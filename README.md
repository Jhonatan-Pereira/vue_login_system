## Getting Started

### Install Dependencies
```sh
npm install -g @vue/cli
```

### Install Packages
```sh
npm install
```

### Development
```sh
npm run serve
```

Clique em [http://localhost:8080](http://localhost:8080) para visualizar no browser.

Projeto [backend](https://github.com/Jhonatan-Pereira/node_login_system).
